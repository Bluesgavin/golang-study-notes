# 条件判断/If

## if 的用法

```go
if expression{
 // do somthing
}
```

如果 expression 为 true，则执行代码块中的逻辑

## if...else...

当然我们还能有多分支

```go
if expression{
 // do somthing
}else {
 // 前面条件不符合
}
```

## if...else if...else

```go
if expression{
 // do somthing
}else if expression {
 // do other thing
}else{
 // 前面条件都不符合
}
```
