# 协程 Concurrency

协程是 Golang 中的一大核心功能，它在底层实现了协程的管理，并提供了API。开发者只需利用API开发即可，不需关心操作系统底层协程的管理实现。

Golang 的协程内容主要有： goroutine 和 channels 。
