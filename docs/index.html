# 📝Go 语言学习笔记

本手册是本人学习 GO 语言的整理资源

GO 语言是由 Google 开发的新型开发语言，主要特点有：

  • 由 Google 团队维护，有强大团队背书
  • 容易上手
  • 优化对并发的处理
  • 日益增长的语言生态

官方地址： https://go.dev/ (opens new window)

# 说明

Go 的官方资源其实很齐全，有十分完整的细节文档和许多不同的快速上手教程。但体验下来，我觉得还是差点意思，主要是课程还是太简短，很多东西无法深入。而完整的文档又太冗长。所以我想打造一个可以让想学习 Go 的人可以真正上手掌握这门语言的小册子。

虽然官方用一些小案例作为教材，确实可以让学习者有学下去的兴趣。但其实不掌握语法基础，在看代码时仍会有很多不懂。所以我的册子思路还是跟大部分语言一样，从数据类型、语法开始，再到实际的项目开发，阅读者只要按照菜单顺序阅读即可。
